# Nir-Ohana.github.io

Portfolio website hosted on GitHub Pages.

## Local structure
- The homepage is [index.html](index.html)
- Styling is in [styles.css](styles.css)
- Background graphics are in [three-bg.js](three-bg.js) (Three.js via CDN ES module)

## Publish with GitHub Pages
1. Push to your GitHub repo (this repo).
2. In GitHub: **Settings** → **Pages**.
3. Under **Build and deployment**:
	- **Source**: “Deploy from a branch”
	- **Branch**: `main` (or your default branch) and `/ (root)`
4. Wait for the deployment to finish; GitHub will show the site URL.

If you prefer the `/<repo>` URL under your username, keep the repo name as `<username>.github.io`.
